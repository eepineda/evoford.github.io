Stores App - Mobile First

Use Case: Store staff quickly requests waste collection, tracks status, and views history.

Main Screens:

  • Login / Sign-Up: Fast access via store ID or QR onboarding.
  • Scan QR Code: Opens camera to scan bin/waste-type QR code.
  • Request Collection Form: Auto-fills location and bin type; fields for volume, time window, notes.
  • Request Confirmation: Status “Scheduled” with ETA.
  • Track Collection: Status updates from Requested to Collected.
  • History: Past collections with filters.
  • Settings / Help: Support chat, FAQ, profile management.

Business Management - Admin Portal

Designed for: Operators and planners with desktop-first UI and map interface.

Main Screens:

  • Login / Admin Dashboard: KPIs, pending requests, alerts.
  • Collection Requests Overview: Approve, decline, reschedule.
  • Schedule Planning: Calendar and route optimization.
  • Driver Assignment Panel: Drag-drop or dropdown assignment.
  • Live Map View: Real-time driver locations.
  • Driver Performance: Efficiency metrics.
  • Client Management: Contact info, collection history.
  • Settings / Support / User Management.

Drivers App - Mobile Experience

Designed for: Quick daily use, mobile-only app for drivers.

Main Screens:

  • Login / Daily Start: Driver login and start of work day.
  • Today’s Route Overview: Map and list view with optimized routes.
  • Collection Task Detail: Address, waste type, notes.
  • Update Status: Mark tasks as Arrived, Collected, or Issue Reported.
  • Map Navigation: GPS integration or simulated deep link.
  • End of Day Summary: Completed tasks, notes, and metrics.
