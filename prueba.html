Stores App - Mobile First

Use Case: Store staff quickly requests waste collection, tracks status, and views history.

Main Screens (5–7 screens):

  • Login / Sign-Up: Fast access, store ID or QR onboarding.
  • Scan QR Code (Home Screen): Opens camera to scan bin/waste-type QR code.
  • Request Collection Form: Auto-fills location and bin type. Fields: volume, time window, notes.
  • Request Confirmation: Status set to “Scheduled” with estimated collection time.
  • Track Collection: Status updates: Requested → Accepted → In Transit → Collected.
  • History: List past collections with filters by date and waste type.
  • Settings / Help: Support chat, FAQ, and profile management.

Business Management - Admin Portal

Designed for: Operators and planners. Desktop-first with map interface.

Main Screens (6–8 screens):

  • Login / Admin Dashboard: KPIs, pending requests, system alerts.
  • Collection Requests Overview: Approve, decline, reschedule requests.
  • Schedule Planning: Calendar with route optimization logic.
  • Driver Assignment Panel: Drag-and-drop or dropdown assignment interface.
  • Live Map View: Real-time driver locations, active collections.
  • Driver Performance: Metrics on time efficiency and completed pickups.
  • Client Management: Contact info and collection history per store.
  • Settings / Support / User Management.

Drivers App - Mobile Experience

Designed for: Quick daily use, mobile only.

Main Screens (4–6 screens):

  • Login / Daily Start: Driver login and work day start.
  • Today’s Route Overview: Map + list view with optimized route.
  • Collection Task Detail: Address, waste type, special notes.
  • Update Status: Options to mark: Arrived, Collected, Issue Reported.
  • Map Navigation: GPS integration or deep link simulation.
  • End of Day Summary: Completed collections, notes, and metrics.
