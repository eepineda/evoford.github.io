EvoSustain - Smart Waste Collection Apps

Innovative solutions for stores, drivers, and business managers

1. Stores App

Use Case: Store staff quickly requests waste collection, tracks status, and views history.

Main Screens

  • Login / Sign-Up with store ID or QR onboarding
  • Scan QR Code to identify bin/waste type
  • Request Collection Form: auto-filled location & bin type, volume, time window, notes
  • Request Confirmation with scheduled status and ETA
  • Track Collection with status updates (Requested → Accepted → In Transit → Collected)
  • History of past collections with filters
  • Settings / Help with support chat and FAQs

2. Business Management (Admin Portal)

Designed For: Operators and planners. Desktop-first with map interface.

Main Screens

    Login / Dashboard with KPIs and system alerts
  • Collection Requests Overview (approve, decline, reschedule)
  • Schedule Planning with calendar and route optimization
  • Driver Assignment Panel (drag-and-drop or dropdown)
  • Live Map View of drivers and active collections
  • Driver Performance metrics (efficiency, pickups)
  • Client Management (contacts, collection history)
  • Settings, Support, User Management

3. Drivers App

Designed For: Quick daily use, mobile only.

Main Screens

  • Login / Daily Start
  • Today’s Route Overview (map + list, optimized)
  • Collection Task Detail (address, waste type, notes)
  • Update Status (Arrived, Collected, Issue Reported)
  • Map Navigation (GPS integration or simulation)
  • End of Day Summary (completed tasks, notes, metrics)
